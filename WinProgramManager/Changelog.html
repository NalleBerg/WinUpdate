WinProgramManager Version 2026.01.19

New in this version (19 January 2026, 09:00 GMT+1):

  • Initial Release: First public release of the database management component
  • Automated Database Updates: WinProgramUpdater automatically synchronizes package database with winget repository
  • Silent Background Operation: Updates run without visible windows, suitable for scheduled tasks
  • Complete Update Pipeline: Fetches packages, manages additions/removals, filters invalid IDs, queries tags
  • Tag Inference System: 45 name-based patterns for automatic categorization of packages
  • Correlation Analysis: Smart tag suggestion based on 66.67% similarity threshold
  • Automatic Backup System: Rolling 10-backup system with timestamped database files
  • Configurable Logging: Enable/disable logging via ENABLE_LOGGING flag in header file
  • Relative Log Directory: Logs written to ..\log directory relative to executable location
  • 90-Day Log Retention: Automatic pruning of old log entries to maintain manageable file size
  • Console Test Mode: WinProgramUpdaterTest.exe provides verbose output for debugging
  • Hidden Production Mode: WinProgramUpdater.exe runs silently for scheduled tasks
  • PowerShell Build Scripts: Complete database rebuild and incremental restore scripts
  • SQLite3 Integration: Full SQLite3 database with comprehensive package metadata schema
  • Ignore Tag Database: Separate database for filtering unwanted generic tags
